#An Agent-Based Modeling Approach to Assess Post-Flood Disease Outbreaks Following the 2022 Floods in Pakistan


## Overview

This repository contains the Python implementation of an Agent base model framework (ABM) to represent system heterogeneity and dynamic interactions in studing the case of Pakistan for diseases outbreak simulation. In addition, social vulnerability and the evolving interplay between hazard, exposure, and risk are explicitly incorporated into the modelling framework to provide a more comprehensive understanding of post-flood disease dynamics.

## Model Components

It has information of 

- Household vulnerability indices (Data taken from Pakistan Statistical Bureau (PSB).12 indicators are used in this study
- Flood Exposure
- Food Intensity
- Expected Annual Damage (EAD)-
- Disease transmission
- Policy intervention scenarios and assessment
- Sensitivity Analysis


## Intervention Scenarios

The following three intervention scenarios were integarted and evaulated against no-policy scenario

- No Policy
- Enhanced WASH (Hygiene scenario)
- Emergency Preparedness (Awareness scenario)
- Mobile Clinics (Integrated scenario during floods)


## Model results
